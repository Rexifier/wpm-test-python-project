WPM Speed Test

A simple Python-based Words Per Minute (WPM) Typing Speed Test that measures how fast and accurately a user can type a given text within a fixed time.

This project was created as a computer science / Python assignment and focuses on core programming concepts such as input handling, time measurement, string comparison, and basic performance analysis.
